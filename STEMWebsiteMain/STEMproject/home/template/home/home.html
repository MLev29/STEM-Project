Deeper Look Into Django

Django is a popular python package used in order to make all backend functions for a website. A basic interpretation of how the library works can be seen in the flow chart on the right.


How Does Django Work?

1. User connects to the website, this sends a HTTP request.

2. The http request is processed by a python file called Urls.py which will find the correct page or subpage of that website.

3. After having found the correct page, another python file views.py is responsible for getting that page's HTML, CSS and Javascript Code.

4. While the HTML, CSS and Javascript code is being loaded the views.py file will send another request to a file called models.py this file will load any logic for the web page if there is any.

5. All of this data is sent to the user in the form of the formatted and working web page as a HTTP response.

HTML Code Showcase

In this section, we will introduce basic web programming concepts as shown by the HTML code to the left.

<!DOCTYPE html> - Every html file starts with this line and is followed by a <html> and </html> at the end of the file

<h1> - First header tag, used to display a title, for sub title

<p> - Paragraph tag, used to display blocks of text. <h2>,<h3>... which results in a slightly smaller title

<br> - break tag, this is used to skip a line.

class="" A Class allows us to refer code from a Javascript or CSS file in order to define how an element acts or looks.

style="" Lets us manually edit or introduce CSS code into a selected element



CSS Code Showcase

In this section, we will introduce basic web formatting concepts as shown by the CSS code to the right. There are two ways of implementing CSS either in the header of a HTML file or in another file linked to the HTML using the <link> tag

body {} - formatting will apply for all the code of the website due to most of the code being located inside the body of the code.

h1 {} - Indicates custom formatting for any elements inside of header 1 tags

p {} - Indicates custom formatting for any elements inside of paragraph tags

background-color: - this property lets us edit the background color of a website by inputting a valid color or hex code

color: - refers to text color accepts a color or hex code

Other - there are many other properties which may be use to format a webpage, some of these inclue font-family, font-weight, margin.
