How Does it Work?

1. User connects to the website, this sends a HTTP request.

2. The http request is processed by a python file called `Urls.py` which will find the correct page or subpage of that website.

3. After having found the correct page, another python file `views.py` is responsible for getting that page's HTML, CSS and Javascript Code.

4. While the HTML, CSS and Javascript code is being loaded the `views.py` file will send another request to a file called `models.py` this file will load any logic for the web page if there is any.

5. All of this data is sent to the user in the form of the formatted and working web page as a HTTP response.
